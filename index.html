HTML-elements

The HTML element is everything from the start tag to the end tag: <tagname>Content goes here...<tagname>

HTML Paragraphs

The HTML <p> element defines a paragraph.
A paragraph always starts on a new line,
and browsers automatically add some white space
(a margin) before and after a paragraph.
